The files can be run directyl, e.g. 'python contextFreeGrammar.py' and 'python unificationGrammar.py'
They both output each sentence, the number of parses for that sentence, and each parse

contextFreeGrammar.py contains the code with just rules for the context free grammar
unificationGrammar.py expands on this to contain the features that parse the sentences with more granularity
unificationGrammarExt.py contains some additional rules and features as outlined in the Extensions section of the report
